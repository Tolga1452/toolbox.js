# Contributing

## How to contribute

1. Fork the repository on GitHub and clone it locally.

2. Run `npm install` to install the dependencies.

3. Create a branch for your changes.

4. Once your changes are done, run `npm run build` to build the package.

5. Create tests for your changes in the `__tests__` directory

6. Run `npm test` to make sure all tests pass.

7. If all tests pass, commit your changes and push them to your fork.

8. Create a pull request.
